# Human Computer Interfaces

## UI Design Goals - 8 Principles for usability
- Learnability
- Retention
  - re-establish proficiency, 再次使用这个UI时的印象和熟练度
  - "Memorability"
- Performance
- Error Rates
  - how severe
  - how easily to recover
- Subjective Satisfaction
  - how pleasant to use
- Accessability
- Functionality
  - range of operation
  - "how much it can do"
- Trustability
