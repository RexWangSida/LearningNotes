# SQL(Structured Query Language)
- descriptive words
- non-procedural language

## Data Retrieval
- **Using `SELECT` (columns) and `FROM` (table)**
- Table Example: Products
  | prod_name |
  |-----------|
  | Shampoo   |
  |Toothpaste |
  |Deodorant  |
  |Toothbrush |
```sql
SELECT prod_name
FROM Products;
```
